This module does the tracking and management of all financial inputs and behaviors.

Development Roadmap:

Create models for: Income, Expense, Budget, Category, and Savings Goal.
Build endpoints to:
Log income and expense entries.
Set or update spending limits.
Fetch breakdowns by category, timeframe, or tags.

Add business logic:
Detect overspending and notify ai_module.
Calculate budget usage and projections.
Integrate with analytics module for chart/summary generation.
Build recurring transactions functionality.

Optionally integrate with external banking APIs in future.
