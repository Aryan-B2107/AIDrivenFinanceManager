This module does all the intelligent, automated decision-making and financial behavior analysis.

Development Roadmap:

Build endpoints to handle chat-based queries .

Implement AI logic to adjust environment variables (e.g., budget tightening, or relaxing based on upcoming salary).
Add machine learning hooks later (e.g., classification of spending types, goal prediction, NLP for chat).
