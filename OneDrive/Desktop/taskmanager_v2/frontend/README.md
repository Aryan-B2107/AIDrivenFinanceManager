# 📱 Task Manager Frontend

This is the React Native frontend (built with Expo) for the Task Manager app.

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start Expo:
```bash
npx expo start
```

3. Test on Android/iOS/Browser using QR code or emulator.

## 🧾 Screens

- Task List Screen
- Add Task Screen

Backend API: Django REST (`http://<103.124.140.222>:8000/api/`)
