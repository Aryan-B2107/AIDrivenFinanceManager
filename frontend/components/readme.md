What this module does is store all reusable UI components that can be used across multiple screens — like buttons, cards, headers, chart elements, etc.

Development Roadmap:
Create a Button, TextInput, and Card component for consistent styling.
Build chart components (e.g., pie chart, bar chart) to display financial data.
Make a reusable ChatBubble for the AI interface.
Create a GoalProgressBar to show savings progress.
Add ExpenseItem, TransactionRow, and ScoreBadge for finance and gamification views.
Use props and StyleSheet.create() to make everything customizable.
