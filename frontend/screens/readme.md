What this module does is contain individual full-page views of the app — like Dashboard, Budget, Leaderboard, Chatbot, etc.

Development Roadmap:

Create a Dashboard.js that summarizes income, expenses, AI tips, and scores.
Build a BudgetScreen.js to let users set and view budgets.
Add Leaderboard.js to show top savers based on gamification.
Create a ChatScreen.js to interact with the AI chatbot.
Build a GoalScreen.js where users can set and track goals.
